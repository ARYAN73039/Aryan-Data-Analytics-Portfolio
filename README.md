# Aryan-Data-Analytics-Portfolio
# Retail Sales Intelligence Dashboard

## Project Overview

This project analyzes retail sales performance, customer behavior, product profitability, and regional trends using Power BI.

## Tools Used

* Power BI
* DAX
* Excel

## Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* Total Orders
* Total Customers

## Dashboard Pages

### Executive Dashboard

Provides a high-level overview of sales and profitability.

### Product Analysis Dashboard

Identifies top-performing and low-performing products.

### Customer Analysis Dashboard

Analyzes customer segments and purchasing behavior.

## Business Insights

* Identified top-performing regions and categories.
* Analyzed customer contribution to revenue.
* Highlighted low-profit products requiring attention.
